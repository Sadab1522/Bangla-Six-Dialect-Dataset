# Bangla-Six-Dialect-Dataset
A balanced dataset of 3,600 sentences covering six regional dialects of Bangladesh: Rangpur, Bogura, Sylhet, Chittagong, Pabna, and Cumilla.

## Dataset Structure
- Total sentences: 3,600
- Sentences per region: 600
- Format: CSV/XLSX
- Columns: Sentence, Label
